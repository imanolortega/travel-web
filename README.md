# **Travel Web**

Web de viajes desarrollada con Next.js y componentes personalizados usando Once UI.

## 🧱 Tecnologías

- [Next.js](https://nextjs.org/)
- [Once UI](https://once-ui.vercel.app/)

## 📦 Instalación

```bash
npm install

```

## ▶️ Desarrollo

```bash
npm run dev

```

## 🚀 Build

```bash
npm run build

```
